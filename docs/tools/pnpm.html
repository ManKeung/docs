# pnpmopen in new window

包管理工具

# 安装

npm install -g pnpm

npx add -g pnpm

# 使用

  1. 升级pnpm
pnpm add -g pnpm
  1. 安装
命令含义
pnpm install/i安装package.json依赖
pnpm add sax保存到 dependencies 配置项下
pnpm add -D sax保存到 devDependencies 配置项下
pnpm add -O sax保存到 optionalDependencies 配置项下
pnpm add -g sax安装软件包到全局环境中
pnpm add sax@next安装标记为 next 的版本
pnpm add sax@3.0.0安装指定版本 3.0.0
  1. 更新
pnpm up
pnpm up --latest
pnpm up foo@2
pnpm up "@babel/*"
  1. 删除
pnpm remove <name>
pnpm rm <name>
pnpm uninstall <name>
pnpm un <name>
  1. 配置
pnpm config set store-dir /path/to/.pnpm-store

速度快、节省磁盘空间的软件包管理器
