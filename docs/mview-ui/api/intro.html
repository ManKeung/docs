# 介绍

此ts函数方法，为mView框架提供的一部分功能，它的实现，需要通过ts调用，而不是组件的形式。 工具库中的所有方法，均挂载在$u对象下，调用方法如下：

  • 如果是在ts中，需要通过uni.$m.xxx形式调用，如调用去除空格的trim方法：
console.log(uni.$m.trim('  abc  ')) // 去除两端空格
  • 如果是在元素中，无需前缀uni，如：
<view>
    去除两端空格：{{$m.trim('  abc  ')}}
</view>
