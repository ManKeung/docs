# 介绍

mView

多平台快速开发的UI框架

# 来由

由于Vue3的到来uniapp社区还没有找到合适的UI框架，故此mView应运而生。感谢uViewopen in new window，如果您的项目还是基于Vue2开发的建议使用uView这是个优秀的UI框架，如果后期uView推出Vue3版本UI框架您可以选择使用uView。

关于mView的取名，首字母m来自于min的首字母，uni-app是基于vue.js，view是uni-app中最基础，最重要的组件，故取名mView，表示基于uniapp的小巧的UI框架。

# 适用领域

mView是基于uni-app的UI矿机，uni-app 是一个使用 Vue.js 开发所有前端应用的框架，开发者编写一套代码， 可发布到iOS、Android、H5、以及各种小程序(微信/支付宝/百度/头条/QQ/钉钉)等多个平台(引言自uni-app网)。

注意：mView使用的Vue3和ts开发，不适合nvue，小程序主要兼容微信小程序，其他小程序如有问题可以提出修复。
