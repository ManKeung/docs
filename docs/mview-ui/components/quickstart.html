# 快速上手

首先保证我们已经下载mview-ui相关文件，并且放在项目根目录uni_modulesopen in new window下。

  • main.ts:
import { createSSRApp } from 'vue'
import mView from '@/uni_modules/mview-ui'

export function createApp() {
    const app = createSSRApp(App)
    app.use(mView)

    return {
        app
    }
}

 



 





  • uni.scss:
@import '@/uni_modules/mview-ui/theme.scss';
 
  • App.vue:
<style lang="scss">
@import '@/uni_modules/mview-ui/index.scss';
</style>

 


# 如何使用

通过上面配置在某个页面可以直接使用组件，无需通过import引入组件。

<template>
	<m-icon name="left" :size="size" />
</template>

<script lang="ts" setup>
import { ref } from 'vue'

const size = ref(18)
</script>

 







# 关于uni.$m

mView将$m对象同时挂载到了uni对象上，这意味着您可以在外部的ts文件中，通过uni.$m.xxx的形式去调用mView提供的一些工具方法。
