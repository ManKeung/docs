# 为什么做mView

由于vue3的出现，为了使用新技术，由于uViewopen in new window目前只支持vue2还没兼容vue3，所以mView就出来了。

# 非常感谢uView

mView参考uView，uView是非常优秀的框架，如果你的项目没使用vue3建议使用uView框架，如后期uView兼容了vue3你也可以选择使用uView，也可以继续选择mView。
