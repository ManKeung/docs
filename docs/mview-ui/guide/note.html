# 注意事项

mView使用技术：vue3、ts等新技术，如果你还是不了解建议不要使用mView框架，建议选择其他框架。

本项目主要是便于本人对新技术的了解掌握，以及工作项目的支持。如你也需要欢迎你的使用它。

# "$m"被占用

mView有很多的自带的js工具函数，挂载在$m对象中，而$m又挂载在app.config.globalPropertiesm中，所以，请勿占用$m变量名并挂载到app.config.globalProperties，uni.$m也挂载上。

# mView对nvue的支持

目前没有支持，因为nvue还没支持vue3，所以框架没支持，如你项目需要nvue的支持建议不要使用本框架。

# 技术要点

  • mView依赖SCSS预处理器，所以您需要给HBuilder X安装 “sass/scss编译” 插件
  • mView基于HX的最新版本开发，旧版本可能会不兼容，如遇问题，请升级HX尝试是否可解决
  • mView要求项目开启uniapp的V3版本
  • mView使用ts开发
